Introduction

The four basic mathematical operations include;

  • Addition
  • Subtraction
  • Multiplication
  • Division

    • Understanding them is the key to progressing in mathematics especially Algebra.

    Addition

    In this section, two numbers are added together to form a result called a sum.

    Below is a code that shows how to add two numbers:

    a=2; b=3; c=a+b; cout << "The sum of a and b is "<< c <<;

    The sum of three numbers is also indicated below:

    a=2; b=3; c=5; d=a+b+c; cout << "The sum of a and b and c is "<< d <<;

    Subtraction

    This is the difference between two variables. It is a complementary operation to addition

    Below is a code that shows how to subtract two numbers:

    a=5; b=3; c=a-b; cout << "The difference between a and b is "<< c <<;

    Multiplication

    This is the product between two variables or numbers.

    The word times is also used in the place of the multiplication.

    It is a commutative function.

    Below is a code that shows how to multiply two numbers:

    a=2; b=7; c=a*b; cout << "The product of a and b is "<< c <<;

    Division

    This is the inverse of multiplication.

    The number being divided is called the dividend.

    The number by which it is divided is called the divisor.

    The result of this operation is called a quotient.

    Below is a code that shows how to divide two numbers:

    a=10; b=5; c=a/b; cout << "The quotient of a and b is "<< c <<;

    Conclusion

    This four basic operations can be used to build upon other functions like;

  • square
  • square root
